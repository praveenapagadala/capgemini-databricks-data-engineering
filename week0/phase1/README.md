
# Phase 1 – SQL to PySpark

## Overview
In this phase I practiced converting basic SQL queries into PySpark operations and understanding how data transformations work in Databricks.

## Learned
- SELECT and column operations
- Filtering data using WHERE / filter()
- Aggregations using GROUP BY / groupBy()
- Mapping SQL queries to PySpark transformations

## Implementation
- Wrote SQL queries in sql_queries.sql
- Implemented the same logic in PySpark using pyspark_code.py
- Stored output screenshots in the outputs/ folder

## Challenges
- Understanding how groupBy() works in PySpark
- Writing join operations between multiple datasets
- Converting SQL logic to PySpark syntax
