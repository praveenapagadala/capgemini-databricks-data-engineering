# Phase 4A – Bucketing & Segmentation

## Objective

The objective of this phase was to understand how continuous numerical data can be converted into meaningful categories using bucketing and segmentation techniques.

## What I Implemented

* Calculated total spending per customer
* Applied segmentation using different approaches:
  * Conditional logic (fixed thresholds)
  * Quantile-based segmentation
  * Window-based ranking
* Grouped customers based on segments
* Compared results across different methods

## Key Learnings

* Segmentation helps simplify analysis and supports business decision-making
* Fixed threshold segmentation is simple but may not adapt well to different datasets
* Quantile-based segmentation divides data more evenly
* Window functions help in ranking and dynamic segmentation

## Challenges Faced

* Understanding differences between segmentation methods
* Choosing appropriate thresholds
* Interpreting quantile results

## Conclusion

This phase helped me understand how raw numerical data can be transformed into meaningful categories. Segmentation is widely used in real-world applications like customer analysis, marketing strategies, and business insights.
